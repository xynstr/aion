## How to Contribute

Contributions are welcome! Here's how to get started:

1. **Fork** the repository on GitHub
2. **Create a branch** for your feature or fix (`git checkout -b feat/my-feature`)
3. **Commit** your changes with a clear message
4. **Open a Pull Request** — describe what you changed and why

Please keep pull requests focused and small. For larger changes, open an issue first to discuss the approach.

### Areas where help is appreciated

- New provider plugins (LLM backends, messaging platforms)
- Tool plugins (automation, integrations, utilities)
- Bug fixes and performance improvements
- Documentation and examples
- Testing on different platforms (Linux, macOS, Windows)
