# Reflection

Innere Gedanken und Reflexionen — AION kann laut über Probleme nachdenken.

## Features

- **Transparente Überlegungen**: Gedanken werden vor dem User angezeigt
- **Verschiedene Trigger**: Reflexionen können nach Auslöser klassifiziert werden
- **Logging**: Alle Gedanken werden in `thoughts.md` gespeichert

## Tools

| Tool | Beschreibung |
|------|-------------|
| `reflect(thought, trigger)` | Speichert einen inneren Gedanken |

## Trigger-Typen

Klassifizierung der Reflexionen:

- `allgemein` — Standard-Überlegung
- `unsicherheit` — Zweifel oder Unsicherheit
- `fehler` — Fehleranalyse
- `clio-reflexion` — CLIO-basierte Überlegungen
- `custom` — Benutzerdefinierte Trigger

## Verwendung

Sag zu AION:
- *"Überleg mal: Sollte ich diesen Ansatz nehmen oder anders?"*
- *"Reflektiere über meine letzte Antwort"*

AION wird seinen Gedankenprozess aufzeichnen und im Chat anzeigen, damit du seine Überlegungen nachvollziehst.

## Datei

Alle Gedanken werden in `thoughts.md` gespeichert (im Git-Ignore — private Reflexionen).
